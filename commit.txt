# Commit Message Format

## Structure
<type>(<scope>): <subject>

## Types
- feat:     New feature
- fix:      Bug fix
- docs:     Documentation changes
- style:    Code formatting (no logic change)
- refactor: Code restructuring
- perf:     Performance improvements
- test:     Adding tests
- chore:    Maintenance tasks
- ui:       UI/UX styling changes
- anim:     Animation updates

## Scopes
- init, config, deps, styles
- navbar, footer, hero, about
- capabilities, highlights, gallery, contact
- chatbot, particles, scroll
- components, hooks, utils
